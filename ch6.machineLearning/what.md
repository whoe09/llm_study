## 머신러닝이란?  
Train Data + Anser -> learning -> catch Predict  

### 하이퍼 파라미터 
- 사용자가 직접 정해줄 수 있는 변수
- ex) 뉴런의 수 or 층의 수

### 가중치와 편향과 같은 매개변수
- 모델이 학습하는 과정에서 얻어지는 값

TP: true를 true로 예측한경우(정답)
FP: false를 true로 예측한경우(오답)
FN: true를 false로 예측한경우(오답)
TN: flase를 false로 예측한경우(정답)

### 모델 학습 과정
1. 훈련 데이터, 검증 데이터, 테스트 데이터 분류 (예를 들어 6:2:2)  
2. 훈련 데이터 모델 학습(epoch+1)
3. 검증 데이터로 모델 평가 + 정확도/오차 계산
4. 검증 데이터 오차 증가 = 과적합 징후 -> step5 이동 | while step2
5. 모델 학습 종료 + 테스트 데이터로 모델 평가
